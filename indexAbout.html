Рады приветствовать вас в нашем приложении!

Мы - команда начинающих, любящих своё дело веб-разработчиков. Хотим представить вам наш первый проект.

Данный проект предназначен для создания вопросов по разным темам в IT, которые будут полезны и интересны начинающим специалистам этой сферы деятельности.

У вас есть возможность не только узнать что-то новое для себя, но и самостоятельно создать вопрос, который будет интересен другим пользователям.

Этот проект написан без использования каких-либо библиотек и фреймворков и поддерживает хранение данных в 4 разных форматах:

CSV

JSON

YAML

XML

Наш проект был создан кровью, слезами и молитвами с любовью и при поддержке IT-колледжа Dev Education ❤
